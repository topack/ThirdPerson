Character :
	-Property
		.Name
		.Attributes (health, strength, intelligence, crit chance...)
		.Spells
		.Level
		.buffs/debuffs
		.faction
	-dies when attribute health == 0
Race :
	-Property
		.Name
		.Spells
Class :
	-Property
		.Name
		.Spells
		.ManaType (mana, rage...)
Attribute :
	-Property
		.Name
		.Value
		.Max value
	-Usable by spells to modify their value
	-Value/MaxValue can be changed from outSide
Spell :
	-Property 
		.Name
		.ManaTypeCost
		.Buff/Debuff
		.CastRange
		.Self
		.Value
	-When used do an action : Pop Doodad, apply buff/debuff, do damage
	-Can use character attributes to modify its values

Player GameObject :
	-Have Character
		.Name
		.Attributes (health, strength, intelligence, crit chance...)
		.Spells
		.Level
		.Auras
		.Faction
		.FactionsReputation
	-Have Race
		.Name
		.Spells
	-Have Class
		.Name
		.Spells
		.ManaType (mana, rage...)
	-Have Bags
	-Have QuestsInProgress
	-Properties :
		.Experience
		.ExperienceToNextLevel

Npc GameObject :
	-Have Character
		.Name
		.Attributes (health, strength, intelligence, crit chance...)
		.Spells
		.Level
		.Auras
		.Faction
		.FactionsReputation
	-Have ManaType
	-Have AI
	-Have QuestsToGive
	-Have Loot

Spell GameObject :
	.Name
	.Cost
	.Range (0==self)
	.CastTime (instant == 0, channel < 0, cast > 0)
	.Duration (duration of the spell)
	.Cooldown
	.GlobalCoolDown
	.School (shadow, holy, physical...)
	.Mechanic (Shielded, Polymorphed)
	.DispellType (magic, poison, curse...)
	.Effect (list of other spell, aura, mechanics to do ex :
			-Apply Aura: Mod Damage Done (Physical)
			-Apply Aura: Mod Skill - Temporary (Cooking)Value: 10
			-Apply Aura: Periodic DamageValue: 5 every 3 seconds)
			-Summon Object (176501) Radius: 3 yards
	.Flags (boolean condition ex :
			-Cannot be used while shapeshifted,
			-Cannot be used in combat
			-Passive spell
			-Aura is hidden
			-Starts ticking at aura application)

Quest GameObject :
	.Title
	.LevelRequirement
	.LevelDifficulty (Red, yellow, green, grey. Each difficulty change xp reward)
	.Faction
	.DescriptionText
	.ProgressText
	.CompletionText
	.Objective
	.Sharable
	.QuestsCompleteRequired
	.Rewards (items)
	.Gains (experience and reputation)
